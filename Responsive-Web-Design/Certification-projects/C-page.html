Introduction

C is a high level language introduced by Brian Kerrington and Dennis Ritchie in 1972.
It is a modular programming language adjacent to hardware used, therefore it is used to design system software.
It is a simpler, safe and a portable language.
It is case sensitive i.e(Capital and Small letters are different)

C and C++

Both C and C++ similar syntax, compilation process and code structure.
C is a structural programming language that was used for system applications and low level programming whereas C++ is an object-oriented programming language

Control Structures in C

They direct the flow of execution in the program.
According to ANSI-C programming language there are 3 types of control structures

  1. Decision Control Structure
    • If Statements
    • If...Else Statement
    • Nested if...else Statement
    • Switch Case statement
  2. Looping Control Structure
    • For Loop
    • Do.....While Loop
    • While Loop
    • Nested Loops
  3. Jumping Control Structure
    • Break structure
    • Go to structure
    • Continue structure
If Statement

It is a decision control structure that is concerned or considers the true part of the condition.
If the condition is true, the program must do something and if it is false it stops

				
					if (condition)
					{
						statement 1;
						statement 2;
					}
If...Else Statement

It is a decision control structure that considers both the true and false part of the program

If the condition is true, the program must do something and if it is false it does something else

				
                                        if (condition)
                                        {
                                                statement 1;
                                                statement 2;
                                        }
					else 
					{
						statement 1;
						statement 2;
					}
Nested If Statement

Occurs when an if...else statement is present inside te body of another if or else statement

It is useful when a program has to make a series of related decisions

				
                                        if (condition)
                                        {
                                                statement 1;
                                                statement 2;
                                        }
					else if (condtion)
					{
						statement 1;
						statement 2;
					}
                                        else
                                        {
                                                statement 1;
                                                statement 2;
                                        }
Switch Case Statement

It is used to make unlimited decisions based on the value of a condition i.e it is a simplified version of a nested if...else staement

				
					switch (condition)
					{
						case 1: statement 1;
						case 2: statement 2;
						case 3: statement 3;
						default: statement 4;
					}
While Loop

It is an entry loop i.e the condition is first checked before execeution of the statement

The condition is repeatedly executed while it is still logically true.

				
					while (condition)
					{
						statement 1;
						statement 2;
					}
For Loop

It is a loop that is used to repeat a block of statement a number of times that is known in advance

It is an entry loop i.e the condition is first checked before execeution of the statement

				
					for (initialization; condition; increament/decreament)
					{
						statement 1;
						statement 2;
					}
Do...While Loop

It is an example of an exit loop i.e it executes the statement inside the do block atleast once before checking the condtion

				
					do {
						statement 1;
						statement 2;
					}while (condition);
Nested Loops

It occurs when we have a loop inside another loop

				
					for (initialization; condition; increament/decreament)
                                        {
						for (initialization; condition; increament/decreament)
                                        	{
                                                	statement 1;
                                                	statement 2;
                                        	}
                                        
                                        }
Break Statement

It is mainly used to exit the switch case entirely after each case label

Can also be used to break loops.

					
                                        switch (condition)
                                        {
                                                case 1: statement 1;
							break;
                                                case 2: statement 2;
							break;
                                                case 3: statement 3;
							break;
                                                default: statement 4;
							break;
                                        }
Continue Statement

It transfers control of execution to the beginning or end of the loop.

There are times when you want to stay in a loop but skip over some statement within a loop instead of breaking the loop, we can use a continue statement

When you put the continue keyword in a for loop, execution jumps to the update expression (i++ in the example):

					
						for (var i = 0; i < 3; i++) {
							if (i === 1) {
							continue;
							}
							printf("%d", i);
						}
GOTO Statement

A goto statement is used to 

				
					label name://it is an identifier that tells the goto statement where to jump
						statement 1;
						statement 2;
						statement 3;
					goto label name;
Finite and Infinite Loop

A Finite Loop - A loop that is terminated when the condition is false.
It has either an increament/decreament operator or a break statement

An infinite Loop - An infinite loop is one in which the conditional expression is left out entirely thus is upto the code inside the loop to determine the conditions for ending the loop
